Resultados

A ideia do projeto seria um ambiente pra enviar receitas para uma editora, ou programa de tv, que compartilharia essas receitas com sua audiência.

  • Pagina Inicial

    Primeira página contém uma breve apresentação do site, explicando o seu objetivo e como usar.

    Foi utilizado um banner para chamar a atenção dos visitantes, junto com o nome da empresa fictícia.

    Bem-vindo à YouCook

  • Menu

    A navegação do site se encontra no header, sendo responsivo e colapsando em telas menores que tablets.

    Possui os links para as 4 páginas desenvolvidas, tendo a logo tambem como link para a Inicial.

  • Cadastro

    Foi considerado os dados mais necessários para uma receita, como nome, autor, tempo de preparo e quantidade de porções que rende.

    Para os ingredientes e modo de fazer, foi criado um botão para adicionar mais um campo caso precise, afinal esses dados variam conforme a receita.

    Para cada campo/input foi adicionado verificações de caracteres e todos são obrigatórios.

    No fim do formulário há dois botões, um para limpar todos os inputs e outro para enviar os dados, que só funcionará se os campos tiverem de acordo com os requisitos.

  • Receitas

    A receita envida no cadastro é apresentada nessa página, organizando os dados para fácil leitura.

    Foi utilizado bordas para separar os conteúdos.

  • Sobre

    Conforme interpretação do enunciado, a página Sobre contém informações sobre o site, tendo a apresentação do trabalho, o autor e evidências no tratamentos do elementos.
